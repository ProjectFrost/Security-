1. **Password Hashing:** Protects passwords in storage.
2. **SSL/TLS Encryption:** Secures data in transit.
3. **VPN Encryption:** Safeguards communication over networks.
4. **Data-at-Rest Encryption:** Shields stored data from unauthorized access.
5. **[[Tokenization]]:** Replaces sensitive data with non-sensitive tokens.
6. **[[Digital Signatures]]:** Ensures data integrity and origin verification.
7. **[[Homomorphic Encryption]]:** Allows computation on encrypted data, preserving privacy.
8. **[[Steganography]]:** Conceals data within other media for covert communication.
